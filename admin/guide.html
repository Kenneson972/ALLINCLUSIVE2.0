Tutoriel Pas-à-Pas

1
Ajouter votre première villa

Section "Gestion Villas" → Bouton "Ajouter Villa" → Remplir formulaire complet → Sauvegarder

2
Uploader des images

Section "Galerie Images" → Glisser-déposer photos → Compression automatique → Copier URLs pour villas

3
Configurer les paramètres

Section "Paramètres" → Informations de contact → Sauvegarder configuration

4
Exporter vos données

Dropdown Admin → "Exporter données" → Sauvegarde JSON complète → Archivage sécurisé

Dépannage

  1. Ouvrir Console navigateur (F12)
  2. Vérifier erreurs dans l'onglet Console
  3. Vérifier chemins des fichiers JS/CSS
  4. Recharger la page (Ctrl+F5)
  5. Tester en mode incognito

  1. Vérifier localStorage disponible
  2. Restaurer depuis dernière sauvegarde JSON
  3. Utiliser "Importer données" en cas de perte
  4. Reset complet: localStorage.clear()

  1. Vérifier taille fichier (max 5MB)
  2. Format supporté: JPG, PNG, WebP
  3. Vérifier quota localStorage navigateur
  4. Compresser images si nécessaire
